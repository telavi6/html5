# HTML5 - рЃњрЃљрЃЎрЃЋрЃћрЃЌрЃўрЃџрЃў I: рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃЋрЃћрЃЉрЃњрЃЋрЃћрЃарЃЊрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ

## рЃњрЃљрЃЎрЃЋрЃћрЃЌрЃўрЃџрЃўрЃА рЃЏрЃўрЃЏрЃЮрЃ«рЃўрЃџрЃЋрЃљ
рЃљрЃЏ рЃњрЃљрЃЎрЃЋрЃћрЃЌрЃўрЃџрЃќрЃћ рЃерЃћрЃЋрЃЦрЃЏрЃюрЃўрЃЌ рЃЕрЃЋрЃћрЃюрЃў рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃЋрЃћрЃЉрЃњрЃЋрЃћрЃарЃЊрЃА. рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ **Emmet**-рЃА HTML рЃЕрЃЮрЃюрЃЕрЃ«рЃўрЃА рЃАрЃгрЃарЃљрЃцрЃљрЃЊ рЃерЃћрЃАрЃљрЃЦрЃЏрЃюрЃћрЃџрЃљрЃЊ, рЃерЃћрЃЋрЃфрЃЋрЃџрЃўрЃЌ **title**-рЃА, рЃЊрЃљрЃЋрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃЌ CSS рЃцрЃљрЃўрЃџрЃА, рЃњрЃљрЃЋрЃбрЃћрЃАрЃбрЃљрЃЋрЃЌ CSS-рЃўрЃА рЃЏрЃБрЃерЃљрЃЮрЃЉрЃљрЃА рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃўрЃЌ, CSS-рЃА рЃњрЃљрЃЋрЃљрЃюрЃБрЃџрЃћрЃЉрЃЌ рЃЊрЃљ рЃЏрЃЮрЃЏрЃЊрЃћрЃЋрЃюрЃЮ рЃњрЃљрЃЎрЃЋрЃћрЃЌрЃўрЃџрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА рЃерЃћрЃЋрЃЦрЃЏрЃюрЃўрЃЌ рЃљрЃ«рЃљрЃџ рЃЉрЃарЃћрЃюрЃЕрЃА. рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃБрЃџрЃў рЃљрЃЏрЃЮрЃфрЃљрЃюрЃћрЃЉрЃўрЃА рЃАрЃљрЃюрЃљрЃ«рЃљрЃЋрЃљрЃЊ рЃЏрЃЮрЃАрЃљрЃ«рЃћрЃарЃ«рЃћрЃЉрЃћрЃџрЃўрЃљ рЃњрЃЦрЃЮрЃюрЃЊрЃћрЃЌ live рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ(рЃџрЃЮрЃЎрЃљрЃџрЃБрЃарЃў рЃЋрЃћрЃЉ-рЃАрЃћрЃарЃЋрЃћрЃарЃў), рЃарЃЮрЃЏрЃћрЃџрЃўрЃА рЃЮрЃарЃњрЃљрЃюрЃўрЃќрЃћрЃЉрЃљрЃф рЃЏрЃљрЃарЃбрЃўрЃЋрЃўрЃљ рЃЏрЃарЃљрЃЋрЃљрЃџрЃў рЃ«рЃћрЃарЃ«рЃўрЃЌ, рЃћрЃарЃЌрЃћрЃарЃЌрЃў рЃљрЃарЃўрЃА **VS Code**-рЃерЃў Live5 рЃърЃџрЃљрЃњрЃўрЃюрЃўрЃА рЃЕрЃљрЃДрЃћрЃюрЃћрЃЉрЃљ. рЃљрЃЏрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃерЃћрЃњрЃћрЃФрЃџрЃћрЃЉрЃљрЃЌ рЃърЃарЃЮрЃћрЃЦрЃбрЃерЃў рЃњрЃљрЃЎрЃћрЃЌрЃћрЃЉрЃБрЃџрЃў рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃЏрЃДрЃўрЃАрЃўрЃћрЃарЃљрЃЊ рЃюрЃљрЃ«рЃЋрЃљ.

## р▓юр▓љр▓Љр▓ўр▓»р▓ў 1: `index.html` рЃцрЃљрЃўрЃџрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ
1. рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ **VS Code** (рЃљрЃю рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃАрЃљрЃДрЃЋрЃљрЃарЃћрЃџрЃў рЃарЃћрЃЊрЃљрЃЦрЃбрЃЮрЃарЃў) рЃЊрЃљ рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃАрЃљрЃЦрЃљрЃдрЃљрЃџрЃЊрЃћрЃерЃў.
2. рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃљрЃ«рЃљрЃџрЃў рЃцрЃљрЃўрЃџрЃў рЃАрЃљрЃ«рЃћрЃџрЃўрЃЌ **`index.html`**.
3. рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ рЃћрЃА рЃцрЃљрЃўрЃџрЃў, рЃЕрЃљрЃгрЃћрЃарЃћрЃЌ `!` (рЃФрЃљрЃ«рЃўрЃџрЃўрЃА рЃюрЃўрЃерЃљрЃюрЃў) рЃЊрЃљ рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ  `Tab` рЃљрЃю `Enter`.
   - **Emmet**-рЃў рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃерЃћрЃЦрЃЏрЃюрЃўрЃА HTML рЃЕрЃЮрЃюрЃЕрЃ«рЃА.
4. рЃЕрЃЮрЃюрЃЕрЃ«рЃерЃў, `<title>` рЃЌрЃћрЃњрЃерЃў рЃЕрЃљрЃгрЃћрЃарЃћрЃЌ:
   ```html
   <title>Home</title>
   ```

## р▓юр▓љр▓Љр▓ўр▓»р▓ў 2: `style.css` рЃцрЃљрЃўрЃџрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ рЃЊрЃљ рЃЏрЃўрЃЉрЃЏрЃљ
1. рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃљрЃ«рЃљрЃџрЃў рЃцрЃљрЃўрЃџрЃў рЃАрЃљрЃ«рЃћрЃџрЃўрЃЌ **`style.css`** рЃўрЃњрЃўрЃЋрЃћ рЃАрЃљрЃЦрЃљрЃдрЃљрЃџрЃЊрЃћрЃерЃў.
2. CSS-рЃўрЃА рЃЊрЃљрЃАрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃџрЃљрЃЊ HTML рЃцрЃљрЃўрЃџрЃўрЃА `<head>` рЃњрЃљрЃюрЃДрЃЮрЃцрЃўрЃџрЃћрЃЉрЃљрЃерЃў рЃЕрЃљрЃгрЃћрЃарЃћрЃЌ:
   ```html
   <link rel="stylesheet" href="style.css">
   ```
> **­ЪњА р▓ар▓Ер▓ћр▓Ћр▓љ:** 
> **VS Code**-рЃА Emmet-рЃўрЃЌ рЃљрЃЎрЃарЃўрЃцрЃћрЃЌ `link` рЃЊрЃљ рЃљрЃЏрЃЮрЃўрЃарЃЕрЃћрЃЌ `link:CSS` рЃЊрЃљ рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ `Tab` рЃљрЃю `Enter`.

## рЃюрЃљрЃЉрЃўрЃ»рЃў 3: CSS-рЃўрЃА рЃЏрЃБрЃерЃљрЃЮрЃЉрЃўрЃА рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљ
1. рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ `style.css` рЃЊрЃљ рЃЕрЃљрЃгрЃћрЃарЃћрЃЌ рЃерЃћрЃЏрЃЊрЃћрЃњрЃў:
   ```css
   body {
       background-color: lightblue;
   }
   ```
> **­ЪњА р▓ар▓Ер▓ћр▓Ћр▓љ:** 
> **VS Code**-рЃА Emmet-рЃўрЃЌ `{}` рЃцрЃарЃЕрЃўрЃџрЃћрЃЉрЃўрЃА рЃерЃўрЃњрЃюрЃўрЃЌ рЃљрЃЎрЃарЃўрЃцрЃћрЃЌ `bc` рЃЊрЃљ рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ `Tab` рЃљрЃю `Enter`.
> CSS-рЃЌрЃЋрЃўрЃА Emmit рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА рЃцрЃўрЃњрЃБрЃарЃБрЃџрЃў рЃцрЃарЃЕрЃ«рЃўрЃџрЃћрЃЉрЃўрЃА рЃерЃўрЃњрЃюрЃўрЃЌ, рЃАрЃћрЃџрЃћрЃЦрЃбрЃЮрЃарЃћрЃЉрЃўрЃА рЃљрЃЎрЃарЃћрЃцрЃљ рЃЏрЃЮрЃњрЃўрЃ«рЃЊрЃћрЃЉрЃљрЃЌ рЃ«рЃћрЃџрЃўрЃЌ.

2. рЃерЃћрЃўрЃюрЃљрЃ«рЃћрЃЌ рЃДрЃЋрЃћрЃџрЃљ рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃљ рЃЊрЃљ рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ **`index.html`** рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃерЃў.
3. рЃЌрЃБ рЃњрЃЋрЃћрЃарЃЊрЃўрЃА рЃцрЃЮрЃюрЃў рЃњрЃљрЃ«рЃЊрЃљ рЃфрЃўрЃАрЃцрЃћрЃарЃў, CSS рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА рЃАрЃгрЃЮрЃарЃљрЃЊ.

## рЃюрЃљрЃЉрЃўрЃ»рЃў 4: CSS-рЃўрЃА рЃљрЃюрЃБрЃџрЃўрЃарЃћрЃЉрЃљ (Reset)
рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃћрЃЉрЃўрЃА рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃБрЃџрЃў рЃАрЃбрЃўрЃџрЃћрЃЉрЃўрЃА рЃгрЃљрЃАрЃљрЃерЃџрЃћрЃџрЃљрЃЊ рЃЊрЃљрЃљрЃЏрЃљрЃбрЃћрЃЌ `style.css` рЃцрЃљрЃўрЃџрЃерЃў:
```css
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
} 
```

## рЃюрЃљрЃЉрЃўрЃ»рЃў 5: рЃљрЃ«рЃљрЃџрЃў рЃЉрЃарЃћрЃюрЃЕрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ (Lesson-II)
1. рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ рЃбрЃћрЃарЃЏрЃўрЃюрЃљрЃџрЃў рЃЊрЃљ рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃАрЃљрЃЦрЃљрЃдрЃљрЃџрЃЊрЃћрЃерЃў.
2. рЃњрЃљрЃЊрЃљрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ рЃарЃЮрЃЏ рЃ«рЃљрЃарЃЌ `main` рЃЉрЃарЃћрЃюрЃЕрЃќрЃћ:
   ```sh
   git checkout main
   ```
3. рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃљрЃ«рЃљрЃџрЃў рЃЉрЃарЃћрЃюрЃЕрЃў рЃАрЃљрЃ«рЃћрЃџрЃўрЃЌ **Lesson-II**:
   ```sh
   git branch Lesson-II
   ```
4. рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ рЃљрЃ«рЃљрЃџ рЃЉрЃарЃћрЃюрЃЕрЃќрЃћ:
   ```sh
   git checkout Lesson-II
   ```

## рЃерЃћрЃ»рЃљрЃЏрЃћрЃЉрЃљ
РюЁ рЃерЃћрЃЋрЃЦрЃЏрЃћрЃюрЃўрЃЌ `index.html` **Emmet**-рЃўрЃЌ  
РюЁ рЃерЃћрЃЋрЃфрЃЋрЃљрЃџрЃћрЃЌ **title** рЃАрЃљрЃ«рЃћрЃџрЃўрЃЌ РђъHomeРђю  
РюЁ рЃЊрЃљрЃЋрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЌ **`style.css`**  
РюЁ рЃњрЃљрЃЋрЃбрЃћрЃАрЃбрЃћрЃЌ CSS рЃцрЃЮрЃюрЃўрЃА рЃерЃћрЃфрЃЋрЃџрЃўрЃЌ  
РюЁ рЃњрЃљрЃЋрЃљрЃюрЃБрЃџрЃћрЃЌ рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃўрЃА рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃБрЃџрЃў рЃАрЃбрЃўрЃџрЃћрЃЉрЃў  
РюЁ рЃерЃћрЃЋрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃљрЃ«рЃљрЃџрЃў рЃЉрЃарЃћрЃюрЃЕрЃў **Lesson-II** рЃЏрЃЮрЃЏрЃЊрЃћрЃЋрЃюрЃЮ рЃњрЃљрЃЎрЃЋрЃћрЃЌрЃўрЃџрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА  

­Ъџђ **рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃњрЃљрЃЎрЃЋрЃћрЃЌрЃўрЃџрЃў:** рЃерЃћрЃЏрЃЊрЃћрЃњ рЃњрЃљрЃЎрЃЋрЃћрЃЌрЃўрЃџрЃќрЃћ рЃЋрЃўрЃАрЃгрЃљрЃЋрЃџрЃўрЃЌ рЃЎрЃЮрЃюрЃбрЃћрЃюрЃбрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃўрЃарЃћрЃЉрЃљрЃА рЃўрЃАрЃћрЃЌрЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃЌ, рЃарЃЮрЃњрЃЮрЃарЃћрЃЉрЃўрЃфрЃљрЃљ `<header>`, `<nav>` рЃЊрЃљ `<section>`.

**рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃљ:**
```sh
git add .
git commit -m "Lesson I: рЃАрЃљрЃгрЃДрЃўрЃАрЃў рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ HTML рЃЊрЃљ CSS-рЃўрЃЌ"
git push origin Lesson-II
```
